# Smart Travel Planner

A hybrid machine learning project that predicts travel budget using airfare data combined with trip-level features such as hotel type, season, travel mode, and traveler category.

# Features

* Real airfare dataset training
* Random Forest Regressor
* Hybrid feature engineering
* Smart distance-based duration estimation
* Student budget guidance
* Tourist destination recommendations
* Streamlit interactive UI

# Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

# Model Performance

* Final R² Score ≈ 0.80
